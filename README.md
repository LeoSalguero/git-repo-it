### Configuracion inicial de git

#### Configuro el usuario

```sh

    git config user.name "LeoSalguero" 

```
#### Configuro el mail

```sh

    git config user.email "lsalguero24091311@gmail.com"

```
#### Listo las configuraciones del usuario

```sh

    git config --get-regexp user

```
